# Troylab.

- This project aims to design a set of APIs..

## Perquisites
 - PHP8
 - composer
 - npm

## Getting started
- exec the following into your terminal
    1. cd into project directory
    1. cp .env.example .env
    1. composer install
    1. php artisan key:generate
    1. php artisan migrate --seed
    1. php artisan test
    1. php artisan serve
